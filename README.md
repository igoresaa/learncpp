# fstream reader (archive/fstream.cpp)
Простенькое приложение для чтения текста из файла, также можно записать в конец нужную строку. Использование: ./cpp <имя файла> <параметр> <записываемая строка, если параметр -write>.
Поддерживаемые параметры:
-reset: перезаписать файл.
Полностью стирает файл.

-write: записать строку в конец файла.
Записывает переданную параметром строку в конец файла. Если строка состоит из нескольких слов, используйте "". Пример: ./build/cpp text.txt -write "некоторый текст".
